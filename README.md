📖 Overview

This repository contains a structured collection of Python scripts and Jupyter Notebooks developed to explore and practice core concepts in Python programming, Computer Vision, Image Processing, and Natural Language Processing (NLP).

The work focuses on practical implementation, with an emphasis on real-time applications using OpenCV and introductory integration of AI-based chatbot systems.

📁 Repository Contents

The repository includes a combination of notebooks and Python scripts covering:

Image processing and computer vision workflows
Real-time OpenCV-based applications (camera preview, filters, face detection)
Fundamental Python programming exercises
Basic NLP implementations
Simple AI chatbot integrations
